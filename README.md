## Getting Started

### Prerequisites

- Web server with PHP support (e.g., Apache, Nginx)
- PHP 8.2 or higher
- Database server (MySQL / MSSQL SSMS)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/SimplyMon/BOZRIAH-POS.git
cd BOZRIAH-POS
```
